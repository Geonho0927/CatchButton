# 버튼 잡기 게임 (C# 코딩)

## 개요
- C# 프로그래밍 학습
- 핵심기능: 마우스를 피해 움직이는 버튼을 클릭하여 잡는 게임을 만듦. 버튼을 클릭 or 실패 시 점수가 바뀌며, 실패를 20번 하면 게임오버를 함. 게임오버 후엔 재시작 기능이 있음.
- 화면구성: 폼 제목엔 버튼위치와 점수, 실패 횟수가 표기되며, 화면에는 버튼이 있음.
